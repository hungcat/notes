# はじめに

`docs/memo/` ディレクトリに Markdown ファイルを配置するだけで、日付やタグに基づいて自動的にサイドバーへ整理・追加されます。手動で設定ファイルを更新する必要はありません。

## 執筆方法

ローカル開発モード（`pnpm docs:dev`）では、専用の**メモ入力画面**（エディタ）が利用可能です。ブラウザから直接執筆し、ローカルへの保存や Git への Commit & Push をシームレスに行うことができます。もちろん、エディタを使わずに `docs/memo/` 以下に直接 `.md` ファイルを作成しても構いません。

## ローカルプレビュー

```bash
pnpm i
pnpm docs:dev
```

## ビルド

```bash
pnpm docs:build
```
